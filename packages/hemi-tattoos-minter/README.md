# @hemilabs/hemi-tattoos-minter

A TypeScript package for interacting with the HemiTattoos soul-bound NFT contract.

## Features

- Typed functions for minting Tier 1 and Tier 2 NFTs
- Automatic token approval handling
- Query functions for checking mint status and fetching metadata
- Support for both Hemi mainnet and Sepolia testnet

## Installation

```bash
npm install @hemilabs/hemi-tattoos-minter
```
